# Bewakoof-clone-  **[Deployed Link](https://thriving-mochi-1e7141.netlify.app/)**
Bewakoof is an Online Shopping site for Men and Women Clothing.

## Description
This project is the clone of the frontend website of  Bewakoof-clone. Its all the landing pages from the original working website, with Login/Sign out functionality.This project is completed individually in 5 days.

## Functionalities :
<ul>
<li>Landing Page - Home Page</li>
<li>Login/Sign Up Page</li>
<li>Mens page</li>
<li>Women page</li>
<li>Accessories Page</li>
</ul>
